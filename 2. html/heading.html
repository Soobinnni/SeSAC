시맨틱 태그

시맨틱 태그는 웹 문서의 구조와 의미를 더 명확하게 전달하기 위해 사용되는 HTML 태그들을 말합니다.
이러한 태그들은 웹 페이지의 가독성과 접근성을 향상시키며, ✨검색 엔진 최적화(SEO)✨에도 도움이 됩니다.
몇 가지 대표적인 시맨틱 태그를 알아보겠습니다:

  1. <header>: 웹 페이지나 섹션의 제목이나 소개 등을 포함하는 헤더 영역을 정의합니다.
  2. <nav>: 내비게이션 메뉴를 정의하는 영역입니다.
  3. <main>: 문서의 주요 콘텐츠를 정의합니다. 한 페이지에 하나의 <main> 요소만 사용해야 합니다.
  4. <section>: 문서의 섹션을 정의합니다. 일반적으로 제목(<h1>~<h6> 요소)을 가지며, 관련된 콘텐츠를 그룹화합니다.
  5. <article>: 독립적인 콘텐츠, 뉴스 기사, 블로그 글 등을 정의합니다.
  6. <aside>: 별도의 사이드바 컨텐츠를 정의합니다.
  7. <footer>: 웹 페이지나 섹션의 푸터 영역을 정의합니다.

이 외에도 <figure>, <figcaption>, <time>, <address> 등 다양한 시맨틱 태그가 있습니다. 이러한 태그들을 사용하여 웹 문서의 구조를 명확하게 표현하고 의미를 부여할 수 있습니다.
